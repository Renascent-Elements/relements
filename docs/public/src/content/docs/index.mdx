## Install

<Tabs syncKey="pkg">
  <TabItem label="npm">

```sh
npm install @relements/core
```

  </TabItem>
  <TabItem label="pnpm">

```sh
pnpm add @relements/core
```

  </TabItem>
  <TabItem label="yarn">

```sh
yarn add @relements/core
```

  </TabItem>
  <TabItem label="bun">

```sh
bun add @relements/core
```

  </TabItem>
</Tabs>

Import the stylesheet once at your app entry — that single import gives you light
**and** dark, following the OS automatically:

```js
import "@relements/core/index.css";
```

## A 30-second example

No components to import and no JavaScript to run — style native elements with the
`.re-*` classes:

<Demo src="button.html" name="variants" />

```html
<button class="re-button" data-variant="primary">Save</button>
```

The [Quickstart](/relements/guides/quickstart/) walks through adding a field and
an optional behavior; the [framework guides](/relements/frameworks/) show the
one-line setup for React, Vue, Svelte, and Angular.
