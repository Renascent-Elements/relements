Framework examples

The same one-flow (a re-button, an enhanceTabs region, and a <re-tabs> custom element) built with no wrappers in five stacks.

  • Plain HTML — no build
  • React — run pnpm build:examples first
  • Vue — run pnpm build:examples first
  • Svelte — run pnpm build:examples first
  • Angular — run pnpm build:examples first

See README.md in this folder for setup and caveats.
