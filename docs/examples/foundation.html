Notes

  • Public API: CSS classes (.re-*), data attributes (data-re-*), and CSS custom properties (--re-*).
  • The foundation imports tokens.css, reset.css, and base.css into named cascade layers, so consumer styles always win without specificity hacks.
  • Component styles will register under the re.components layer in later phases.
