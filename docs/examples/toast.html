Toast

Transient notifications via an aria-live region.

Trigger toasts

Notes

  • showToast(message, { tone, duration }) appends a toast to the live region. duration: 0 disables auto-dismiss.
  • Danger tone uses role="alert" for interruptive announcements; others use role="status".
  • If no region exists, one is created and appended to document.body on first call.
