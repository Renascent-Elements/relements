Editor IntelliSense: ship VS Code HTML/CSS custom-data files (`@relements/core/html.custom-data.json`, `@relements/core/css.custom-data.json`) so the `<re-*>` tags, the `data-*` attributes (with their allowed values), the declarative `data-re-*` hooks, and every `--re-*` token autocomplete with hovers. A new [editor setup guide](https://renascent-elements.github.io/relements/guides/editor-setup/) documents the `.vscode/settings.json` wiring and the per-framework TypeScript typings for the custom elements (React, Vue, Svelte, Angular).
